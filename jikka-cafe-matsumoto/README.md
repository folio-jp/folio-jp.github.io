# 実家カフェ｜サンプルHP（営業提案用デモ）

昭和レトロな民家カフェ「実家カフェ」の紹介ページ（スマホ優先の1カラム静的サイト）です。
外部ビルド・フレームワーク不要で、そのまま静的ホスティングに公開できます。

## ファイル構成

```
dist/
├── index.html        エントリーポイント（このファイルを開けば動作）
├── css/
│   └── style.css     全スタイル。配色トークンは :root にまとめてあります
├── js/
│   └── main.js       スクロールフェードのみ（外部ライブラリなし）
└── images/
    ├── hero-room.png ファーストビュー背景・和室席（仮画像）
    ├── parfait.png   季節のパフェ（仮画像）
    └── coffee.png    窓辺の珈琲（仮画像）
```

## ローカル確認

`index.html` をブラウザで直接開くか、任意の静的サーバで配信してください。

```
cd dist
python3 -m http.server 8000   # http://localhost:8000
```

## 公開（Codex での作業想定）

- ルートは `dist/`。`index.html` をトップに配置すればそのまま公開可能です。
- パスはすべて相対参照（`css/`, `js/`, `images/`）。サブディレクトリ公開でもリンク切れしません。
- 外部依存は Google Fonts（Shippori Mincho / Zen Kaku Gothic New）のCDNのみ。
  オフライン/自己ホストが必要な場合はフォントをローカル配置し、`index.html` の
  `<link>` と `style.css` の `font-family` を差し替えてください。

## 差し替え箇所（本番化のとき）

- `images/` の3枚を本番写真に差し替え（比率が変わっても `object-fit` は使っていないため、
  縦横比の近い画像に置き換えると崩れません）。
- 予約フォームは未設置（公式予約URLが不明のため電話導線に集約）。URL確定後にCTAを追加可能。
- 最下部の「営業提案用の非公式デモ」表記は、正式公開時に削除してください。
- `<title>` / `og:title` は現状「実家カフェ｜サンプルHP」。正式名に変更する場合はここ。
